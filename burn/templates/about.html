This service is intended to be used as a way of sharing one-time messages secretly. For example passwords and other sensitive data. I needed such a service when resetting my Mom's email password.

Basic Idea

It works by encrypting the message in javascript using AES using a randomly generated password stretched by scrypt. The message is then stored in memory on the server which creates a uuid as a reference. The client side script creates a special url for you where the key is stored in the anchor part, which is by default not sent to the server by the browser.

Upon receieving the request, the server returns the data referenced by the uuid. The javascript takes the data and decrypts it using the encryptionkey taken from the anchor.

All of the code is opensource and available on github. You are free to reuse as you wish under the MIT license.
